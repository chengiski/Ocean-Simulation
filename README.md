Install 3rd-party packages before building the project

It is recommended to use vcpkg to manage these packages.

https://github.com/microsoft/vcpkg

After configuring Path in Windows:

```cmd
vcpkg install glad
vcpkg install glm
vcpkg install glfw3
// fftw
vcpkg install fftw3
```
